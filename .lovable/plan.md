

This is a question about the project's backend, not an implementation task — no plan needed.

Your project runs on **Lovable Cloud**, which is the backend integrado da Lovable. Você não precisa criar uma conta no Supabase nem gerenciar nada externamente — tudo é gerenciado diretamente dentro do Lovable.

O Lovable Cloud oferece:
- **Banco de dados** (tabelas, migrations, RLS)
- **Autenticação** (login, signup, controle de acesso)
- **Storage** (armazenamento de arquivos e fotos)
- **Edge Functions** (lógica backend serverless)
- **Secrets** (variáveis de ambiente seguras)

Tudo isso está acessível pela aba **Cloud** no seu editor, sem precisar sair do Lovable.
